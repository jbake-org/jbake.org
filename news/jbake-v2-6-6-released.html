JBake v2.6.6 released

I’m happy to announce that JBake v2.6.6 has been released. This is primarily a bug fix release.

You can find the complete changelog for this release here: https://github.com/jbake-org/jbake/milestone/21?closed=1

This release should also be available via Maven Central, SDKMAN and Homebrew over the next few days.

Download JBake v2.6.6 now and get baking!

Fixes

  • Don’t assume document types are valid identifies in OrientDB

  • Custom paths configuration do not work

Enhancements

  • make jetty server context path and host configurable

  • produce reproducible archives (distribution zip and jar files)

Updates/Changes

  • use svg for gitter shield

  • Handle travis-ci.org shutdown

  • publish to sonatype nexus instead of bintray

  • Update gradle, dependencies and plugins

Thanks goes to all those who have taken a moment of their time to open issues or submit contributions.

By Jonathan Bullock - 15 April 2021
